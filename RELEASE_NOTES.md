# Artisan v1.2.3

## Italiano

Questa è una release di consolidamento per **Foundry VTT 14.365**. Non cambia il flusso di gioco principale: rende Artisan più stabile, verificabile e semplice da pubblicare.

### Correzioni principali

- Il backup completo esporta e importa anche le liste **Dissassemblare**.
- Risolto l'errore che poteva interrompere l'esportazione del backup.
- Il bonus competenza strumenti viene applicato una sola volta, anche con più strumenti configurati.
- Manifest, versione, documentazione e pacchetti sono ora allineati.
- Ripuliti codice legacy e asset duplicati.
- Aggiunti controllo TypeScript, validazione traduzioni e generazione automatica delle release.

## English

This is a consolidation release for **Foundry VTT 14.365**. It keeps the main gameplay flow unchanged while making Artisan more stable, verifiable and easier to publish.

### Main fixes

- Full backups now export and import **Disassembly** lists.
- Fixed the error that could interrupt full backup export.
- Tool proficiency is applied only once, even when several tools are configured.
- Manifest, version, documentation and packages are now aligned.
- Removed legacy code and duplicate assets.
- Added TypeScript checks, translation validation and automatic release generation.
