# Artisan - Universal Crafting Suite

Artisan è un modulo per Foundry VTT v14 dedicato a crafting, raccolta, harvest e professioni dei personaggi.

## Funzioni principali

- Crafting con ricette, ingredienti, strumenti e output da Item mondo o compendi.
- Requisiti di professione e livello per le ricette.
- Qualità dell'output basata sul margine del tiro.
- Foraging con risorse casuali miste, peso, quantità min/max, biomi e strumenti.
- Harvest con parti, rarità, rischio rovina, strumenti richiesti e consumo strumenti opzionale.
- Professioni salvate sul PG con XP, livelli e moltiplicatori.
- Import/export ricette, Foraging, Harvest e backup completo Artisan.
- Registro attività e pacchetti predefiniti.

## Compatibilità

- Foundry VTT: v14
- Verificato: build 14.364

## Installazione manuale

1. Scarica `artisan-v1.1.0.zip` dalla release GitHub.
2. Estrai la cartella `artisan` dentro `Data/modules/` di Foundry.
3. Riavvia Foundry e abilita il modulo nel mondo.

## Installazione tramite manifest

Usa il manifest della release GitHub:

```text
https://github.com/Crazy22N/Artisan-Universal-Crafting-Suite-for-Foudry-VTT/releases/latest/download/module.json
```
