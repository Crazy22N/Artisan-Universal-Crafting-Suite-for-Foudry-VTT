# Artisan v1.2.0

## Italiano

Artisan v1.2.0 è la nuova release stabile del modulo, basata sulla serie di sviluppo 1.1.x. Questa versione consolida Creazione, Raccolta, Caccia, Dissassemblare, professioni, qualità degli oggetti e gestione degli strumenti in una build più pulita e pronta per GitHub.

### Novità principali

- Costo monetario configurabile per ogni ricetta.
- Opzione per decidere, ricetta per ricetta, se il costo viene consumato anche in caso di fallimento.
- Requisito della copia della ricetta nell'inventario del PG per poter creare.
- Professioni ufficiali: Erborista, Alchimista, Fabbro, Cacciatore, Conciatore, Cuoco, Minatore, Boscaiolo, Artigiano e Sarto.
- Rarità/probabilità individuale per le risorse di Raccolta, Caccia e Dissassemblare.
- Rimozione del rischio rovina separato dalla Caccia.
- Scroll e posizione della finestra preservati dopo clic, salvataggi e aggiornamenti.
- Documentazione italiana e inglese riscritta e aggiornata.

### Note tecniche

- Foundry VTT minimo: v14.
- Verificato su Foundry VTT 14.364.
- Il modulo non è vincolato tecnicamente a un singolo sistema nel manifest, ma alcune funzioni avanzate sono ottimizzate per D&D 5e o sistemi con struttura dati simile.

## English

Artisan v1.2.0 is the new stable release of the module, based on the 1.1.x development series. This version consolidates Crafting, Gathering, Hunting, Disassembly, professions, item quality and tool handling into a cleaner GitHub-ready build.

### Main highlights

- Configurable monetary cost for each recipe.
- Per-recipe option to consume the cost even when crafting fails.
- Requirement for the character to own a copy of the recipe before crafting.
- Official professions: Herbalist, Alchemist, Blacksmith, Hunter, Tanner, Cook, Miner, Woodcutter, Artisan and Tailor.
- Individual rarity/probability for Gathering, Hunting and Disassembly resources.
- Removed the separate Hunting spoilage mechanic.
- Window scroll position is preserved after clicks, saves and updates.
- Italian and English documentation rewritten and updated.

### Technical notes

- Minimum Foundry VTT version: v14.
- Verified on Foundry VTT 14.364.
- The module is not technically locked to a specific game system in the manifest, but some advanced features are optimized for D&D 5e or systems with a similar data structure.
