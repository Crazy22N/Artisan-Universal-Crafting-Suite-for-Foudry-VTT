# Artisan — Descrizione modulo / Module Description

## Italiano

**Artisan** è una suite per Foundry VTT dedicata a creazione, raccolta, caccia, dissassemblaggio e progressione professionale. Il GM può costruire ricette e liste usando Item o Actor provenienti dal mondo e dai compendi, configurando quantità casuali, rarità, strumenti, livelli, costi e qualità del risultato.

Le professioni incluse sono Erborista, Alchimista, Fabbro, Cacciatore, Conciatore, Cuoco, Minatore, Boscaiolo, Artigiano e Sarto. Artisan è installabile come modulo generico, mentre competenza strumenti, monete, quantità inventario e modifica automatica delle formule sono ottimizzate per D&D 5e o strutture dati simili.

## English

**Artisan** is a Foundry VTT suite for crafting, gathering, hunting, disassembly and profession progression. GMs can build recipes and lists with world or compendium Items and Actors, configuring random quantities, rarity, tools, levels, costs and output quality.

Included professions are Herbalist, Alchemist, Blacksmith, Hunter, Tanner, Cook, Miner, Woodcutter, Artisan and Tailor. Artisan can be installed as a generic module, while tool proficiency, currency, inventory quantities and automatic formula changes are optimized for D&D 5e or similar data structures.
