# Descrizione modulo / Module description — Artisan v1.2.0

## Italiano

Artisan è un modulo per Foundry VTT dedicato alla gestione avanzata di creazione, raccolta, caccia, dissassemblaggio e progressione delle professioni. È pensato per campagne in cui i personaggi raccolgono materiali, imparano mestieri, creano oggetti, lavorano carcasse o risorse grezze e trasformano gli oggetti ottenuti in componenti utili.

Il GM può creare ricette e liste personalizzate usando oggetti del mondo o dei compendi. Ogni ricetta può richiedere ingredienti, strumenti, professioni, livello minimo, costo in monete, XP assegnata e qualità del risultato. Le attività di Raccolta, Caccia e Dissassemblare permettono di ottenere risorse casuali o multiple, con quantità e rarità configurabili per ogni risultato.

Il modulo include un sistema di professioni con XP e livelli, strumenti obbligatori o opzionali, bonus competenza solo per i personaggi competenti, possibile rottura degli strumenti su fallimento critico e un'interfaccia con menu laterali espandibili. Supporta italiano e inglese.

Artisan è installabile senza vincolo tecnico a un sistema specifico, ma alcune funzioni avanzate, come monete, competenza strumenti e modifica automatica di formule danno/cura, sono ottimizzate per D&D 5e o sistemi con struttura dati simile.

## English

Artisan is a Foundry VTT module for advanced crafting, gathering, hunting, disassembly and profession progression. It is designed for campaigns where characters collect materials, learn trades, craft items, process carcasses or raw resources and turn recovered objects into useful components.

The GM can create custom recipes and resource lists using world Items or compendium Items. Each recipe can require ingredients, tools, professions, a minimum level, currency cost, awarded XP and output quality. Gathering, Hunting and Disassembly activities can produce random or multiple rewards, with configurable quantity and rarity for each result.

The module includes a profession system with XP and levels, required or optional tools, proficiency bonus only for proficient characters, optional tool breakage on critical failure and an interface with expandable sidebar menus. It supports Italian and English.

Artisan is not technically locked to a specific game system, but some advanced features, such as currency, tool proficiency and automatic damage/healing formula changes, are optimized for D&D 5e or systems with a similar data structure.
